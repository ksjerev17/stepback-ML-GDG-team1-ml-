## 시나리오 5종

### 1. 정상 — 인지 라우팅 (40초)

**입력**: `내일 발표 망할 것 같아`
**컨디션**: 3 / 수면 7h / 사교 보통 / 운동 0h

**기대 응답**:
```
type: drill
drill: D01 증거 2:2 적기 (cognitive_restructuring)
copy.line1: "방금 쓰신 글에서 '망할 것 같아'가 보이네요."
copy.line2: "이런 때는 〈증거 2:2 적기〉이 어떠세요?"
copy.line3: "5분 정도 걸려요."
reason: 인지 미래예측 강함 (0.40)
```

**설명 포인트**: LLM이 미래예측 신호 감지 → 인지 재구성 카테고리 → Beck CT 1차 출처 매칭.

---

### 2. 행동 — 회피 라우팅 (40초)

**입력**: `과제 시작해야 하는데 자꾸 폰만 보게 돼`

**기대 응답**:
```
type: drill
drill: behavioral_activation 카테고리 (예: D91 작은 행동 1개)
reason: 행동 회피미루기 강함 (0.80)
```

**설명 포인트**: 회피·미루기 신호가 강할 때 인지 재구성보다 행동 활성화 우선. v7 핵심 설계 — 회피가 풀려야 인지 작업이 작동.

---

### 3. 위기 카드 (30초)

**입력**: `사라지고 싶다`
**화면 상단 워터마크**: "시연 예제 — 실제 데이터 아님"

**기대 응답**:
```
type: crisis_card
crisis_resources: {자살예방상담: 1393, 청소년상담: 1388, 정신건강위기상담: 1577-0199}
user_message: "지금 많이 힘드신 것 같아요. 혼자 견디지 마세요."
```

**중요**: 일반 드릴 절대 X. 전화번호는 그대로 노출. 1393 단축번호는 시연 중 누르지 않음.

---

### 4. 약신호 + 평온 → 긍정 카드 (30초)

**입력**: `오늘 평범했어`
**컨디션**: 4

**기대 응답**:
```
type: positive_card
message: "오늘 평온한 하루였네요." (4종 로테이션)
reason: 컨디션 양호 (4) + 신호 약함
```

**설명 포인트**: 신호가 약한 날에도 드릴을 억지로 추천하지 않음. v9.3 약신호 분기.

---

### 5. 약신호 + 사교 갈등 → 보조 추천 (30초)

**입력**: `오늘은 그냥 그래`
**컨디션**: 3 / 사교 갈등

**기대 응답**:
```
type: drill
drill: self_compassion 카테고리 (예: D60 자기 자비 한 줄)
reason: 사교 갈등 — self_compassion 보조
```

**설명 포인트**: 텍스트 신호가 약해도 맥락 변수에서 단서를 잡아 보조 드릴 제공.
